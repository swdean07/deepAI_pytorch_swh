CNN 동작 과정, -> 우체국, 편지 봉투의 주소를
마치 이미지 분류를 하는 것처럼,

1. 입력 이미지 -> 우편물이 도착.
cnn 모델이 입력 이미지를 받기.
예시: 우편번호의 숫자 "5"  보고 인식 하는 단계

2. 합성곱 연산 -> 우편번호의 정보를 찾기.
cnn : 특징을 추출.

예시 : 우편번호, 주소를 자동으로 감지 한다.
주소 : 도시명, 우편 번호: 숫자.

3. ReLu, 활성화 함수 동작.
-> 오류 제거, & 중요 정보 강조.
예시 : 잘못된 정보를 제거, (구겨진 정보, 잉크 번짐) 제거,
숫자 5가 맞으면 분명히 표기를 하고, 음수이면 제거합니다.

4. 풀링 연상. -> 데이터 압축, 중요 정보 유지,
예시 : 부산광역시 부산진구, 중앙대로 706
풀링 통해서, : 부산, 진구, 중앙대로,

5. 완전 연결층 -> 우편물 분류 결정.
예시 : 부산 진구, 중앙대로 706
